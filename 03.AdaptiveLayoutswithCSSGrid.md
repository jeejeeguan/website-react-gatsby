#### 1.更改 VS Code 项目语言设置

React 的 JSX 是 JavaScript 的语法扩展，在 VS Code 中并没有默认开启，需要手动开启，点击底部状态栏的 **SelectLanguageMode** 可以为当前文件选择对应的语言，也可以通过更改 **ConfigFileAssociationFor** 来把 JS 关联为 react。
